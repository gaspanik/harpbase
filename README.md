# Harp Base

## Getting started

1. Install Harp.
2. Clone this repository.
3. Start `harp server`.

## Structure

```shell
├── README.md
├── _bower
│   ├── bootstrap
│   ├── font-awesome
│   └── jquery
├── bower.json
├── harp.json
└── public
    ├── 404.jade
    ├── _data.json
    ├── _layout-inner.jade
    ├── _layout.jade
    ├── _less
    ├── _partial/
    ├── about.jade
    ├── css/
    ├── fonts/
    ├── images/
    ├── index.jade
    └── js/
```

* harp.json: Global meta data.
* _data.json: Local meta data.
